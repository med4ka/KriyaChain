## ⚡ Key Features
* **Cinematic Apple-Style UI:** High-fidelity scroll-driven animations, 3D Parallax Tilt cards, and immersive transitions.
* **True Offline PWA (Progressive Web App):** Operates seamlessly without internet using Advanced LocalStorage Caching & Service Workers.
* **Web Speech API Integration:** Real-time Text-to-Speech for cultural philosophy storytelling.
* **Hardware Integration:** Native QR Code Scanner and iOS/Android Native Share Sheet API.
* **Robust Backend:** Built with Golang featuring Rate Limiting, Connection Pooling, and real-time ownership transfers.

## 🛠️ Architecture
- **API:** RESTful structure with strict `gin-gonic` middlewares.
- **Database:** PostgreSQL synchronized via `gorm` with auto-migration.
- **State Management:** React hooks + Native Browser Storage.
